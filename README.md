This project builds a small data pipeline to collect and analyze technology deals from the eBay Global Tech Deals page. A Selenium scraper (scraper.py) was developed to scroll the page and extract product information including timestamp, title, price, original price, shipping details, and item URL, storing the results in ebay_tech_deals.csv. GitHub Actions was configured to run the scraper automatically every three hours to continuously update the dataset. The raw data was then cleaned using clean_data.py by removing currency symbols, handling missing values, converting prices to numeric format, and calculating the discount percentage. Exploratory Data Analysis was performed in a Jupyter notebook (EDA.ipynb) to analyze deal frequency over time, price distributions, discount patterns, shipping options, and common keywords in product titles. Some challenges included handling dynamically loaded page content and inconsistent price formatting. Possible improvements include scraping additional pages, expanding the text analysis, and storing the dataset in a database instead of CSV.
